# I built my own text editor 'Maho'  !

Maho is a simple text editor, simpler than nano :P for minimalistic people.

Using "termios" library as my main inspiration, i tried to built maho.

*Maho's features*:
``

    [x] Text viewer

    [x] Text editor

    [x] Save after editing

    [x] Save as

    [x] Search in file

    [x] Recursive search

``
